## CSV File Handling

Includes programs for:
- Writing data using 'writerow()' and 'writerows()'
- Reading CSV files using 'csv.reader()'
- Appending data to CSV files

---

## JSON File Handling

Includes programs for:
- Reading JSON using 'json.load()'
- Converting JSON data into CSV format using 'DictWriter'

---

## Text File Handling

Includes programs for:
- Writing text files
- Reading text files
- Appending text files

---

## with open() vs Manual open()

- 'with open()' → Automatically closes file (safe and recommended)
- 'open() + close()' → Requires manual closing
